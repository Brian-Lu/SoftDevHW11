Using div classes in Bootstrap

Bootstrap Columns

Much like foundation, bootstrap uses 12 grid slots. Use the col-xs-# (you can substitute xs for other sizes like sm, md, lg) class to a div to create columns. The # chooses the columns it'll take up.

Bootstrap relies on containers

Bootstrap relies on containers! They utilize the 12 unit grid system and properly wrap website elements. The default class is .container and will create a fixed width container. To create a container that will span the length of the viewport, use .container-fluid.
